# Screenshots

Bu klasör JS Docker Platform'un ekran görüntülerini içerir.

## Dosyalar

- `dashboard.png` - Ana dashboard görünümü
- `containers.png` - Container yönetim sayfası
- `images.png` - Image listesi sayfası
- `build.png` - Interactive build sayfası
- `demo.gif` - Platform demo videosu

## Nasıl Ekran Görüntüsü Alınır

1. Web dashboard'u aç: http://localhost:3000
2. İlgili sayfaya git
3. Ekran görüntüsü al
4. Bu klasöre kaydet
5. README.md'de referans ver

## Demo GIF Oluşturma

1. Screen recorder kullan (OBS, Loom, etc.)
2. Container oluşturma sürecini kaydet
3. GIF formatına çevir
4. `demo.gif` olarak kaydet
